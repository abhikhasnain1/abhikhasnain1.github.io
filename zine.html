Procedural Narrative Bot

Overview

This project is a generative sci-fi horror engine built with Tracery and Twine, deployed via a custom Twitter bot. Drawing on cyberpunk tropes from Gibsonâ€™s Neuromancer, it weaves haunting micro-narratives in real time.

Key Features

  • Hand-crafted grammar rules for thematic consistency
  • Dynamic sentence assembly with embedded randomness
  • Automatic tweet scheduling and interaction tracking
  • Twine interface for user-driven branching experiences

Technical Details

Languages: JavaScript, Tracery, Harlowe (Twine)
Hosting: Node.js on Heroku, integrated with the Twitter API
Data: Custom JSON lexicons for sci-fi vocab, horror motifs

Process Notes

I iterated on grammar constraints to balance coherence with unpredictability. Each rule set was play-tested in Twine before integration into the bot.
